# Неофициальное переработанное расписание РГППУ

Этот Userscript изменяет внешний вид и функционал сайта с расписанием РГППУ (`rfvyhn.rsvpu.ru`). Направлен на более современный интерфейс и повышение удобства использования.
Данный Userscript не является официальным и не связан с оригинальными правообладателями сайта. Сайт с расписанием РГППУ принадлежит их правообладателям.

## Основные возможности

*   **Обновленный Material Design:** Полная переработка интерфейса в стиле Material You (Material Design 3) с использованием библиотеки MDUI.
*   **Адаптивность:** Улучшенный адаптивный дизайн.
*   **Тёмная тема:** Автоматическое переключение между светлой и тёмной темой в зависимости от настроек системы.
*   **Переработанная навигация:** Два стиля на выбор в настройках:
    *   **Классический:** Выдвижное боковое меню (`Navigation Drawer`).
    *   **Новый:** Более современный и удобный, автоматически показывает компактную боковую панель с иконками на десктопе (`Navigation Rail`) и удобную нижнюю панель на мобильных устройствах (`Navigation Bar`).
*   **Улучшенные компоненты:** большинство элементов интерейса (кнопки, диалоги, переключатели, календарь) заменены на современные.
*   **"Бесконечная" прокрутка:** Возможность подгружать расписание на следующие дни без перезагрузки страницы (включается в настройках).
*   **И многое другое:** Разделители недель, установка как PWA-приложение (с оговорками) и другие мелкие улучшения и исправления.

## Установка

Для работы скрипта требуется расширение для браузера — менеджер Userscript'ов.

### Шаг 1: Установите менеджер скриптов

Выберите расширение для вашего браузера. Рекомендуется **Violentmonkey** (тестировалось именно с ним), но Tampermonkey также должен работать.

#### **Для десктопных браузеров:**

*   **Mozilla Firefox:**
    *   [Violentmonkey](https://addons.mozilla.org/firefox/addon/violentmonkey/)
    *   [Tampermonkey](https://addons.mozilla.org/firefox/addon/tampermonkey/)
*   **Microsoft Edge:**
    *   [Violentmonkey](https://microsoftedge.microsoft.com/addons/detail/violentmonkey/eeagobfjdenkkddmbclomhiblgggliao)
    *   [Tampermonkey](https://microsoftedge.microsoft.com/addons/detail/tampermonkey/iikmkjmpaadaobahmlepeloendndfphd)
*   **Google Chrome (и другие браузеры на основе Chromium, в т.ч. Яндекс Браузер):**
    *   [Tampermonkey](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)
    *   **Важно для пользователей Tampermonkey (v5.3+):** После установки может потребоваться [вручную включить выполнение скриптов и включить параметры разработчика для расширений](https://www.tampermonkey.net/faq.php#Q209).
    *   *[Violentmonkey](https://chromewebstore.google.com/detail/violentmonkey/jinjaccalgkegednnccohejagnlnfdag) может быть недоступен для установки в Chrome Web Store из-за перехода на Manifest V3.*

#### **Для мобильных браузеров на Android:**

Некоторые мобильные браузеры поддерживают установку расширений.

*   **Firefox:** Через меню браузера можно устаносить **Tampermonkey** или **Violentmonkey**.
*   **Microsoft Edge:** Через меню браузера можно устаносить **Violentmonkey** или **Tampermonkey**.
*   **Яндекс Браузер:** Поддерживает установку расширений из Chrome Web Store. Инструкция аналогична десктопной для Chrome.

### Шаг 2: Установите Userscript

После установки менеджера скриптов, просто **[Установите Userscript, нажав сюда](https://github.com/StrangeVitec/rsvpu-reworked-schedule-rfvyhn/raw/refs/heads/main/rsvpu-reworked-schedule-rfvyhn.user.js)**.

Менеджер скриптов автоматически откроет новую вкладку, где вам нужно будет просто нажать кнопку **"Установить"** или **"Confirm installation"**, или подобную.

## Обновления

Скрипт будет **обновляться автоматически**. Менеджер скриптов будет периодически (обычно раз в день) проверять наличие новой версии на GitHub.
